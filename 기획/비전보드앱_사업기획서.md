## 1. 문제 정의와 기회

- 비전보드는 자기계발 시장에서 검증된 행동(끌어당김/시각화)이지만, 실제로는 **만들기 귀찮고, 만들어도 안 보게 되는** 게 문제다.
- 종이 비전보드·핀터레스트 콜라주는 "남의 사진"이라 몰입이 약하다. **내 얼굴이 들어간 순간 '남의 장면'이 '내 미래'가 된다.**
- 하루에 가장 많이 보는 화면은 잠금화면(평균 수십~수백 회 해제). **잠금화면을 점유하면 '매일 보기'라는 비전보드의 핵심 행동이 자동으로 해결**된다.
- 자기계발 콘텐츠는 원래 간증(성공 사례) 기반으로 퍼진다 — 『시크릿』, 『잇 웍스』, 미라클모닝 전부 간증 마케팅. 구조적으로 UGC 바이럴과 궁합이 좋다.

## 2. 핵심 제품 (MVP 기능 정의)

### 2.1 만들기 — 얼굴합성 비전보드
1. 원하는 장면 사진 선택(수상 무대, 슈퍼카, 합격증, 몸매, 내 집, 여행지…)
2. 내 셀피 업로드 → AI 얼굴합성으로 "이미 이룬 나"의 실사 이미지 생성
3. **It Works 양식 오버레이**: `이름` + `한 줄 상황(완료형 선언문)` + `날짜`
   - 예: "김하원 — 2026.12.24 첫 단독 콘서트 매진시켰다"
   - 문장은 **과거완료형 선언**을 기본 템플릿으로 강제(『잇 웍스』 방식) — 앱이 어미를 자동 교정해주는 마이크로 기능
4. 폰 화면비(잠금화면/홈화면/위젯) 기준으로 자동 크롭·타이포 배치 → **"엄청 잘 보이는" 상태가 디폴트**

### 2.2 매일 보기 — 잠금화면 점유
- 원탭 배경화면 설정(iOS 잠금화면 위젯 / Android 라이브 배경화면)
- 여러 비전보드 로테이션, D-day 카운트다운 표시
- 아침 첫 잠금해제 시 "오늘도 비보" 확인 탭 → 스트릭(연속일수) 적립

### 2.3 자랑하기 — 피드 (오늘의집 모델 이식)
- 내 비전보드를 피드에 공개(얼굴 모자이크/공개 선택)
- 오늘의집의 "이 가구 어디 거예요?" = 우리의 **"이 템플릿/사진 뭐예요?" → 원탭 '따라 만들기'**
  - 남의 보드에서 배경사진·레이아웃·문구 템플릿만 가져오고 얼굴은 내 걸로 → **따라하기가 곧 신규 생성**
- 격려 전용 리액션(응원/기도/불꽃), 같은 목표끼리 모이는 목표별 탭(수능, 오디션, 다이어트, 내집, 창업…)

### 2.4 간증하기 — It Works 피드 (핵심 바이럴 자산)
- 이뤄졌을 때 "이뤄짐" 버튼 → **비포(비전보드+작성날짜) / 애프터(실제 인증샷+실현날짜)** 2컷 카드 자동 생성
- "작성일로부터 D+137일 만에 이룸" 자동 스탬프 → 이 카드가 인스타/틱톡 공유용 포맷
- 간증 피드는 앱의 랜딩이자 신규 유저 전환 페이지("이게 진짜 되네?")

## 3. 네이밍/브랜딩 — 행동의 대명사화

- 목표: "비전보드했다"(설명형)가 아니라 **"비보했다"** 처럼 동사로 쓰이는 고유명사. (구글링하다, 배민하다, 슬랙보내다 류)
- 메인 후보: **비보 (VIBO)** — "비전보드"의 축약이면서 발음이 짧고, "비보했다 / 오늘 비보 바꿈 / 비보 이뤄짐" 활용이 자연스러움
  - 주의: '비보(悲報, 슬픈 소식)' 동음이의 리스크 → 표기를 VIBO/비보✨ 등으로 시각 고정, 오히려 밈("비보인데 낭보임")으로 역이용 가능
- 보조 후보: 이룸(이룸했다), 됐다(됐다 등록함), 위시샷, 드림록(DreamLock)
- 파생 용어 세트를 처음부터 설계: 비보하다(만들다) / 비보 걸다(잠금화면 설정) / 비보 터졌다(이뤄짐) / 비보 간증
- **상표·앱스토어·도메인·SNS 핸들 선점을 네이밍 확정 즉시 진행** (액션 아이템)

## 4. 콘텐츠 소싱 — "맘에 드는 사진" 문제 (핵심 병목 #1)

사용자가 원하는 건 대부분 ①셀럽/무대 ②럭셔리 라이프 ③특정 장소인데, 이게 저작권·초상권 지뢰밭이다. 소스를 4계층으로 구조화한다.

| 계층 | 소스 | 재가공(합성) 가능성 | 비고 |
|---|---|---|---|
| 1. AI 생성 배경 | 자체 파이프라인으로 생성한 장면 라이브러리 | ◎ 가장 안전 | "음방 1위 트로피 무대", "한강뷰 거실" 등 **수요 상위 장면을 자체 생성해 공식 라이브러리화** — 품질 통제 + 법적 클린 |
| 2. 정식 무료 스톡 | Pixabay, Pexels, CC0/퍼블릭도메인 | ○ | **Unsplash는 주의**: 라이선스상 유사·경쟁 서비스용 사진 대량 수집(컴파일) 금지 조항 있음 → 라이브러리 구축용으론 부적합, 개별 라이선스 검토 필요 |
| 3. UGC 사진 마켓 | 유저가 찍은 사진 업로드 + 재사용 허락 | ◎ (약관으로 확보) | "네 여행사진이 남의 꿈이 된다" — 업로더에게 포인트/수익 셰어. **사용자끼리 사진을 제공하는 네트워크 자산**이자 커뮤니티 참여 동인 |
| 4. 유저 직접 업로드 | 본인이 가져온 사진(사적 이용) | △ 책임 고지 | 피드 공개 시에만 저작권·초상권 필터링. 셀럽 사진 합성은 비공개라도 차단 정책(§5) |

- 초기 90일은 **계층 1(자체 AI 라이브러리)을 메인**으로: 헤드핀 니치의 상위 50개 장면만 고품질로 만들면 충분하다. 소스 문제를 "많이"가 아니라 "좁고 깊게"로 푼다.
- 계층 3은 커뮤니티가 형성된 뒤(3~6개월 차) 오픈 — 오늘의집이 유저 집사진으로 카탈로그를 만든 것과 동일한 수순.

## 5. 얼굴합성 — 기획 원칙과 보안·법무 정책 (핵심 병목 #2)

### 5.1 제품 원칙 (딥페이크 논란과의 구조적 절연)
1. **본인 얼굴만**: 합성 대상 얼굴은 실시간 셀피 촬영(라이브니스 체크)으로만 등록 — 갤러리에서 타인 사진 업로드로 얼굴 등록 불가
2. **타인·셀럽 차단**: 유명인 얼굴 감지 시 합성 거부, 배경 사진에 얼굴이 포함되면 자동 블러/교체
3. **표시 의무**: 모든 합성 결과물에 비가시성 워터마크 + 공유용 카드에 "AI 합성" 표기 — 2026년 시행 AI기본법의 AI 생성물 표시 의무 대응(시행령 기준 법무 검토 필요)
4. **미성년 보호**: 만 14세 미만 가입 차단, 미성년 얼굴 합성 별도 제한 정책
5. **성적·폭력적 장면 원천 차단**: 배경 라이브러리 큐레이션 + 업로드 이미지 NSFW 필터

### 5.2 데이터 보안 정책
- 얼굴 이미지는 **개인정보보호법상 민감정보(생체인식정보) 취급을 전제로 설계**: 별도 명시 동의, 수집 최소화
- 원본 셀피는 합성 처리 후 **즉시 삭제**(또는 온디바이스 처리 우선 검토), 얼굴 임베딩은 암호화 저장 + 탈퇴 즉시 파기
- 합성 결과물의 소유권은 사용자, 서버 보관은 사용자 선택제
- 제3자 얼굴합성 API 사용 시 데이터 국외 이전·재학습 금지 조항 계약 필수 (자체 모델 전환 로드맵 병행)
- 위 항목은 출시 전 **개인정보 전문 법무 검토 1회 필수** (액션 아이템 — 법 조항·시행령은 반드시 최신 기준 재확인)

## 6. 바이럴 루프 설계 (요소 총정리)

```
만들기(얼굴합성+It Works 양식)
   → 잠금화면 설정 (매일 노출 = 리텐션)
   → 자랑 피드 공유 (오늘의집형)
   → 타인의 "따라 만들기" (= 신규 생성, 템플릿 확산)
   → 이뤄짐 → 간증 카드 (비포/애프터 + D+일수)
   → 외부 SNS 공유 → 신규 유저 유입 → 만들기 …
```

바이럴 장치 체크리스트:
- [ ] 잠금화면 스크린샷 자체가 예쁜 홍보물이 되도록 디자인 (Locket 학습)
- [ ] 간증 카드 하단 워터마크 "OO도 비보했다 → vibo.app"
- [ ] 따라 만들기 카운트 노출("이 보드 1,204명이 따라함") — 오늘의집 스크랩수와 동일한 사회적 증거
- [ ] 100일 비보 챌린지(스트릭) + 친구 초대 시 합성 크레딧 지급
- [ ] 목표별 랭킹/명예의전당(이번 달 간증 모음)
- [ ] 격려 네트워크: 같은 목표 유저 매칭, 응원 리액션, "네 사진 내 꿈에 쓸게" 사진 기부

## 7. 볼링핀 전략 (Go-to-Market)

원리: 전체 시장이 아니라 **가장 절박하고 밀집된 니치 하나(헤드핀)를 쓰러뜨려 옆 핀으로 연쇄 확산**.

### 7.1 헤드핀: 아이돌 지망생·오디션 준비생 (+ 그 팬덤)
- 선정 이유: ① 꿈이 시각적으로 극명함(무대·1위·데뷔) = 얼굴합성 효과 극대화 ② 커뮤니티 밀집(학원·연습생 SNS) ③ 간증이 뉴스가 됨 ④ 팬덤으로 자연 확산
- **트리거 시나리오(예시)**: 신인 그룹 멤버가 음악방송 1위 소감에서 "매일 비보했어요(연습생 때부터 1위 비전보드를 잠금화면에)"라고 말하는 순간 — 이 한 방이 카테고리를 연다. (예: 리센느급 신인이 대박 나는 케이스)
  - 실행: 기획사·연습생 학원 제휴로 초기 시딩, 데뷔조·오디션 프로그램 참가자에게 무료 프로 계정, "1위 무대 장면" 전용 템플릿 팩 선제작
- 헤드핀 성공 지표: 오디션/지망생 커뮤니티 내 침투율, 셀럽 언급 1건 이상

### 7.2 핀 확산 순서 (인접성 = 같은 '간증 문화' 보유 집단)
1. **팬덤**: "최애가 쓰는 앱" → 팬들이 자기 꿈으로 전이 (덕질 목표: 콘서트 티켓팅 성공 비보 등 밈화)
2. **수험생/고시생**: 합격증·캠퍼스 합성, D-day와 결합 (시즌성 폭발: 수능·시험 발표일 간증 대란)
3. **다이어트/바디프로필**: 비포/애프터 문화가 이미 존재 — 간증 포맷 즉시 호환
4. **재테크/내집마련·창업**: 30대 확장, 유료 전환율 최고 구간
5. (해외) K-pop 팬덤 경유 동남아·남미 — 한국어 간증 포맷 그대로 수출

### 7.3 시딩 콘텐츠
- 출시 전 『잇 웍스』·시크릿류 독서 커뮤니티, 미라클모닝·갓생 인플루언서 30명에게 선공개
- "비보 간증" 해시태그 챌린지를 앱 출시와 동시에 개시(초기 간증은 팀이 직접 리크루팅한 베타 유저 실사례로)

| B2B/제휴 | 오디션 학원·다이어트 프로그램·재수학원 단체 계정, 목표 달성 브랜드 콜라보(예: 합격 간증 × 교육 브랜드) | 12개월+ |

- 원가 주의: 얼굴합성 추론 비용이 변동비의 대부분 → 무료 티어는 저해상도+횟수 제한으로 설계, 잠금화면(정적 이미지)은 원가 0.

## 9. 핵심 지표 (North Star)
