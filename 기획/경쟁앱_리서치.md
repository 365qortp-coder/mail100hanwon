# 경쟁앱 & 기술 리서치 — 비전보드 · 투두 · 가치관 문답 · 얼굴 라이브니스 (2026-07-22 실측)

> 웹 리서치 3건 병렬 실행 결과 요약. 수치는 대부분 서드파티 추정 또는 업체 자체 주장 — 의사결정용 방향 참고치로만 사용.

---

## 1. 결론 먼저 — 시장 공백 (우리 기회)

1. **"보는 것"과 "하는 것"이 분리된 시장**: 비전보드 앱엔 실행(투두) 기능이 없고, 투두 앱엔 시각화·감성이 없다. 둘을 잇는 앱이 글로벌·국내 모두 부재.
2. **얼굴합성 비전보드는 일회성 툴에 머묾**: FutureSelf.ai류는 이미지 뽑고 끝(일회성 과금) — '미래의 나'를 매일의 투두·위젯·간증으로 연결한 앱은 없음.
3. **한국 시장은 완전 공백**: 국내 비전보드 전용 앱 자체가 없음(수요는 인스타 템플릿/굿노트/캔바가 흡수 중). 마이루틴·루티너리는 루틴 실행에만 집중.
4. **위젯/잠금화면은 검증된 리텐션 무기**: Locket(DAU 900만+, 연매출 추정 $13.5M), Motivation 명언 위젯(월 추정 $70만) — 이 고빈도 접점에 '내 얼굴이 들어간 목표 이미지'를 올린 앱은 없음.
5. **과금 결합 여지**: 이미지 생성(건당) + 구독(AI 코칭/위젯) 하이브리드가 가능하나 경쟁사는 단일 모델에 머묾.
6. **핵심 질문의 답**: "비전보드 + AI 투두 분해 + 얼굴합성" 3종을 모두 갖춘 앱은 **확인되지 않음**. 최근접은 MyFutureSelf(AI 아바타+90일 로드맵+AI 멘토)이나 실사 얼굴합성·잠금화면 전략은 확인 불가.

## 4. 잠금화면 / 위젯 점유 앱

- **Locket**: 친구 사진이 홈 위젯에 실시간 표시. 누적 9,100만 설치, DAU 900만+, 연매출 추정 $13.5M, 2024 흑자 전환 — **위젯 = 하루 수십 번 노출되는 초고빈도 접점**임을 증명.
- **Motivation – Daily Quotes**: 명언/확언 위젯만으로 월 40만 DL, 월 추정 매출 $70만 — 텍스트 확언 위젯 시장의 크기 증명. 약점: 개인화·목표 연동 없음.
- **포토위젯/WidgetClub**: 국내 "갓생" 꾸미기 수요 흡수 중이나 목표관리와 미연결.

## 5. 가치관 문답 — 튤립(2ulip) & 유사 서비스

### 튤립, 가치관 소개팅 (2ulip, ㈜튤립, 2018~)
- **50개 객관식 가치관 질문** (LOVE/WORK/EAT/PLAY/THINK 5 카테고리) + 자기소개 3개(각 600자).
- 핵심 메커니즘: 50개 중 **"나에게 중요한 질문 5개를 내가 직접 선택"** → 그 5개에 같은 답을 한 상대를 매칭. **중요한 질문을 스스로 고르는 행위 자체가 가치관 표명** — 우리 온보딩에 이식할 핵심.
- UX: 사진을 마지막에 공개(가치관 먼저) — "외모보다 가치관"이라는 브랜드 신뢰 형성.
- 사용자 호평 이유: 상대 생각을 미리 알 수 있음, 매칭 후 대화 소재가 이미 있음.

### 매일 문답 패턴 (리듬 벤치마크)
- **썸원(SumOne)**: 커플에게 **매일 1문답**, 내가 답해야 상대 답 공개. 누적 600만+ — 매일 1질문 리듬의 최강 사례.
- **질문일기/투유/HARU Q**: 하루 1질문 저널링, 1년 후 같은 질문 재제시로 "답의 변화" 보여주기 — 가치관 변화 추적 장치로 이식 가능.
- **OkCupid**: 질문마다 ①내 답 ②상대에게 원하는 답 ③중요도 가중치 수집 — 정교한 가중치 모델의 원조.
- 공통 패턴: ①질문 카테고리화 ②중요 질문의 사용자 선택 ③공개 조건 설계 ④답변 누적의 기록 가치.

## 6. 얼굴 라이브니스 (은행앱식 상하좌우 인식) — **기술적으로 가능**

- **명칭**: Active Liveness Detection (지시 응답형) / 표준 ISO/IEC 30107-3.
- **MVP 자체 구현 (권장)**: 구글 **ML Kit Face Detection(무료·온디바이스)** 이 고개 회전 각도(Euler X/Y)·눈 깜빡임·미소 확률 제공 → "고개를 왼쪽→위로" 랜덤 챌린지를 수일~수주에 구현 가능. Flutter 오픈소스 패키지도 존재. iOS는 ARKit(TrueDepth 심도)로 보강 가능.
- **성장기 상용 전환**: AWS Rekognition Face Liveness — **검사당 $0.015(약 20원)**, Amplify SDK로 통합 며칠. 국내 eKYC는 알체라(금융결제원 은행권 공동시스템 사업자)/자회사 useB — 가격 비공개(영업 문의).
- **한계 인지**: 자체 챌린지는 사진·캐주얼 도용 억제용. 사전 녹화 영상 조합·3D 마스크·실시간 딥페이크+카메라 주입은 못 막음(주입 공격은 ISO 표준 범위도 벗어남). 우리는 금융앱이 아니므로 **MVP는 ML Kit 수준 + 등록얼굴-라이브얼굴 대조 + 사후 모니터링**이면 충분, 어뷰징 규모가 커지면 상용 passive liveness로 격상.

## 7. 우리 설계에의 반영 (액션)

1. 온보딩 = 튤립식 "중요 질문 직접 선택" + 썸원식 "매일 1문답" 결합 (기획서 §2.0 반영 완료)
2. Dreamfora의 4단 분해 UX를 직접 써보고 강약점 테어다운 (다음 스프린트)
3. MyFutureSelf 계정 만들어 실제 기능 범위 확인 — 최근접 경쟁자 상시 추적
4. 얼굴 등록 = ML Kit 상하좌우 챌린지로 MVP 확정 (기획서 §5.1 반영 완료)
5. Locket/Motivation의 위젯 온보딩 플로우 캡처해서 벤치마크 문서화
